# ScheduleDaily


## 项目简介

这是一个基于原生 HTML5 + CSS3 + JavaScript 开发的日历任务管理 H5 应用，专为移动设备设计，支持任务的增删改查、批量操作、导入导出等功能。

## 核心功能

### 1. 日历视图
- 月份日历展示，左右滑动切换月份
- 今天的日期特殊标记（橙色圆圈显示"今"字）
- 有任务的日期标记（蓝色圆点）
- 任务全部完成的日期标记（绿色圆点）

### 2. 任务管理
- 点击日期查看当天任务列表
- 点击右下角"+"按钮添加任务
- 支持批量添加任务（每行一个任务）
- 点击复选框切换任务完成状态
- 滑动或点击删除按钮删除任务
- 完成的任务显示删除线和灰色文字

### 3. 区间选择（批量操作）
- 长按日期开始区间选择
- 点击导航按钮切换月份
- 长按另一个日期完成区间选择
- 弹出批量操作面板：
  - 批量添加任务
  - 批量标记完成
  - 批量删除任务

### 4. 数据导入导出
- 导出：复制 JSON 格式的任务数据
- 导入：粘贴 JSON 数据导入任务
- 本地存储：使用 localStorage 自动保存

### 5. 定位功能
- 点击"今天"按钮快速跳转到当前日期

## 使用说明

### 添加任务
1. 点击日历上的日期查看当天任务
2. 点击右下角"+"按钮
3. 在弹窗中输入任务内容（支持多行，每行一个任务）
4. 点击"确认"添加

### 完成任务
- 点击任务左侧的复选框切换完成状态

### 删除任务
- 点击任务右侧的"×"按钮删除任务

### 批量操作
1. 长按起始日期开始区间选择
2. 点击导航按钮切换到目标月份（如需要）
3. 长按结束日期完成选择
4. 在弹出的面板中选择操作：
   - 批量添加：输入任务内容
   - 批量完成：将选中日期的所有任务标记为完成
   - 批量删除：删除选中日期的所有任务

### 导入导出
- 点击顶部"导出"按钮复制任务数据
- 点击顶部"导入"按钮粘贴任务数据进行导入

## 技术实现

- 前端框架：原生 HTML5 + CSS3 + JavaScript
- 数据存储：localStorage 本地存储
- 响应式设计：适配各种移动设备屏幕
- 交互方式：支持触摸和鼠标操作

## 项目结构

```
/index.html    - 主页面结构
/style.css     - 样式文件

```

## 使用方式

直接在浏览器中打开 index.html 文件即可使用，无需安装任何依赖。

推荐使用 Chrome、Safari、Firefox 等现代浏览器访问。
